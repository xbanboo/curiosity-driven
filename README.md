# Curiosity drives us to discover more

> What if we looked at the world from a completely different angle?

A collection of five interactive visualizations that explore unconventional connections between science, history, and imagination. Built with React, Three.js, and D3.

## Projects

**Civilization Mortality** — Treating civilizations as living organisms: using actuarial science and survival analysis on 37 dead civilizations to model how long ours might last.

**Consciousness Transit** — Tracing consciousness across 500 million years of host migrations. Hosts die, consciousness persists — it just jumps to the next carrier.

**History in π** — Mapping pivotal moments in human history onto the digit sequence of π. Coincidence or destiny?

**Protein → Architecture** — Translating protein secondary structures into architectural forms. Helices become towers, beta-sheets become slabs, coils become corridors. Synchronized 3D comparison view with real PDB data from RCSB.

**CMB Typography** — The cosmic microwave background is the universe's first photograph, taken 380,000 years after the Big Bang. Its temperature fluctuations shaped today's galaxies. Now, use it to write.

## Tech Stack

- **React 18** + **Vite** — Fast dev & build
- **Three.js** — 3D protein and architecture rendering with synchronized dual-viewport
- **D3** — Data-driven visualizations
- **RCSB PDB API** — Live protein structure data

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
```

Static output in `dist/` — deploy to any static host (GitHub Pages, Vercel, Netlify).
